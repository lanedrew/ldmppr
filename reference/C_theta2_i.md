# calculates c_theta

calculates c_theta

## Usage

``` r
C_theta2_i(xgrid, ygrid, tgrid, data, params, bounds)
```

## Arguments

- xgrid:

  a vector of grid values for x.

- ygrid:

  a vector of grid values for y.

- tgrid:

  a t value.

- data:

  a matrix of data.

- params:

  a vector of parameters.

- bounds:

  a vector of bounds for time, x, and y.

## Value

returns the product.
