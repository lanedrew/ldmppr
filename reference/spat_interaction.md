# calculates spatial interaction

calculates spatial interaction

## Usage

``` r
spat_interaction(hist, newp, params)
```

## Arguments

- hist:

  a matrix of points (x,y), n x 2.

- newp:

  a new point vector (x,y), length 2.

- params:

  a vector of parameters (alpha2, beta2).

## Value

calculated probability of new point.
